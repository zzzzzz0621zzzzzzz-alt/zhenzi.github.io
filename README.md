# 蔡臻晗个人作品集

这是一个展示我的作品和项目的个人网站，包含HTML网页、图片和视频作品。

## 项目内容

- `可以直接插入图片了.html` - 主要作品集网页
- `index.html` - 首页
- `videoworks/` - 视频作品文件夹
- `相关图片/` - 图片资源文件夹
- `数据新闻/` - 数据新闻项目文件

## 技术特点

- 响应式设计，支持移动端
- 深色/浅色主题切换
- 现代化UI设计
- 包含图片和视频展示

## 使用方法

1. 克隆或下载此仓库
2. 打开 `可以直接插入图片了.html` 或 `index.html` 查看作品集
3. 所有资源都已包含在仓库中

## 部署

可以直接部署到GitHub Pages或其他静态网站托管服务。
